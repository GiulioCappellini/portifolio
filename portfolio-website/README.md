# Portfolio Website

Professional portfolio website developed to present my projects, technical skills and software engineering journey.  
This application serves as the main interface for showcasing my work, experiments and long-term technical evolution as a developer focused on backend engineering and scalable systems development.

---

## Overview

The Portfolio Website was created to centralize projects, organize technical content and provide a professional presentation of my development experience.  
The project focuses on combining modern interface design with scalable architecture principles, allowing continuous expansion as new projects and technologies are added.

---

- Responsive modern interface
- Dynamic project showcase
- Organized project navigation
- Interactive UI components
- Mobile-friendly experience
- Expandable architecture
- GitHub and social integration
- Scalable structure for future backend integration
