Updates:

  • Customize profile

    In the future, it will be possible to change your profile name and image, as well as other options such as reminder preferences, dates, and various other settings.

  • Add email and phone number

    In the future, it will be possible to add an email address and a phone number, which will be used for several purposes. These include verifying your identity and recovering credentials if you forget them, as well as other more targeted features such as reminders and scheduling memories with the same title, similar to Google Photos.

  • Customize login and password

    In the future, it will be possible to change your current login and password, allowing users greater flexibility and easier access to their profile on the site.

  • Edit content

    In the future, it will be possible to edit site content — photos, videos, texts, dates, etc. — as well as add other content, profiles, and additional features.

  • Leave messages

    In the future, a new feature will be implemented that allows users to leave messages for each other, increasing interaction within the site and enabling greater emotional connection between users.

  • Add online chat

    In the future, a private online chat system between users will be implemented, with the ability to create groups of two or more participants. It will also be possible to send links to specific memories, screens, or images within the site.

  • Add AI

    In the future, an interactive AI will be added to assist users in finding specific memories or correcting text. It will also be possible to ask where a setting is located or how a feature works. The AI will be designed to explain how the site works in a simple and accessible way across its different aspects.

  • Add database

    In the future, a database will be added that, in addition to enabling other updates, will make the site’s content dynamically editable.

  • Turn into an app

    In the future, this website will be converted into an Android and iOS application, available for easy download on the appropriate online stores. It will be easier to access, use, and edit, improving the overall user experience.
