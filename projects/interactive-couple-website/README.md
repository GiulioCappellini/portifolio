# Interactive Couple Website

Web platform for couples to store and organize memories, feelings, important dates, and personal content.

## Status

Initial version (MVP / Prototype)

- CRUD functionalities under development
- Page structure defined
- Basic interface incomplete

## Planned functionalities

- Creation and management of sections
- Category-based organization
- Dynamic add, edit, and delete system
- Unified experience without requiring external apps

## Technologies

- HTML
  - Page structure rendered in the browser
- CSS
  - Styling and animations
- JavaScript
  - Application logic
  - Dynamic content generation
  - UI interactions and state handling

## Structure

- /images_videos → images, videos, illustrations, and icons
- /pages → site sections
- /scripts → JavaScript logic
- /styles → CSS styles
